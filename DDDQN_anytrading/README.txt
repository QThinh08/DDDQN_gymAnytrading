Code này được chạy trên jupyter notebook
Mở bằng jupyter notebook rồi run file Untitle.ipynb
Nếu chạy trực tiếp bằng IDLE của python thì chỉ cần run file main.py

Code này sử dụng môi trường gym_anytrading nên hãy bỏ qua file StockTradingEnv.py
và folder data

network.py: chứa cấu trúc của model
replay_memory: chứa experience replay buffer (memory)
agent.py: chứa khởi tạo của agent(network, store_transition,...) nhằm để train
main.py: là hàm chính để run.

Hiện tại code này vẫn đang lỗi liên quan đến multi-dimensional indexing array.
